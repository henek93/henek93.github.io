Политика конфиденциальности

Дата вступления в силу: 16 июля 2025 года

1. Общие положения

Приложение "LinguaLink" (далее — «Приложение») обязуется соблюдать конфиденциальность пользователей. Настоящая политика описывает, какие данные мы собираем, как они используются, и какие меры мы предпринимаем для их защиты.

2. Какие данные мы собираем

Приложение может собирать следующие данные:

  • Имя и адрес электронной почты (при входе через Google);
  • Уникальный идентификатор пользователя (UID);
  • История изучения слов, коллекций и чатов;
  • Выбранные книги и прогресс чтения;
  • Вводимые пользователем сообщения для генерации ИИ-ответов;
  • Устройство, язык системы, версия Android;
  • Рекламный идентификатор устройства (Google Advertising ID).

3. Как мы используем данные

Собранные данные используются для:

  • Аутентификации и синхронизации с облаком Firebase;
  • Анализа использования приложения для его улучшения;
  • Показа персонализированной рекламы через Яндекс SDK;
  • Ответов от ИИ-собеседника через OpenRouter API;
  • Хранения пользовательского прогресса.

4. Сторонние сервисы

Для работы Приложения используются следующие внешние сервисы:

  • Firebase (Google LLC) — аутентификация и база данных;
  • Yandex Mobile Ads SDK (Yandex LLC) — показ межстраничной рекламы;
  • OpenRouter API (https://openrouter.ai) — обработка сообщений с ИИ, включая возможную передачу контента на сторонние языковые модели;
  • Jsoup — для локального парсинга EPUB-файлов (без сети);
  • SharedPreferences — локальное хранение пользовательских настроек и прогресса.

5. Передача данных

Мы не продаём и не передаём персональные данные третьим лицам, за исключением случаев:

  • Обработки сообщений пользователя через OpenRouter API с целью генерации ответов;
  • Аналитики и рекламы с помощью Firebase и Яндекс SDK;
  • По требованию закона или суда.

6. Защита данных

Мы используем технические и организационные меры для защиты данных от утечки, несанкционированного доступа и уничтожения. Все соединения шифруются (HTTPS), доступ ограничен аутентификацией через Firebase.

7. Права пользователя

Вы имеете право:

  • Запросить копию или удаление своих данных;
  • Отозвать согласие на обработку данных;
  • Обратиться с жалобой по вопросу конфиденциальности.
